OPEN PROTOCOL STANDARD

Apps generate tasks.
Any AI runs them.

AIP is an open format for software that doesn't need to embed AI. Your app produces a structured task packet. The user runs it with whatever AI they already have. No API keys. No costs. No lock-in.

EXAMPLE — document_summary task packet
=== AI_TASK ===
version: 1.1
task:    document_summary

input:
  text: [USER_INPUT]
  ... paste any document here ...
  [/USER_INPUT]

rules:
  3_sentences_maximum
  plain_language
  no_preamble

output_format: text
=== END_TASK ===
Paste this into ChatGPT, Claude, Gemini, or any local model. Get a result. Done.
