# Release Notes — AIP v1.1

AIP v1.1 is the current public draft of the Artificial Intelligence Prompt Protocol.

## What AIP does

AIP defines a structured way for software to generate AI tasks that users can execute with their own preferred AI tools, instead of forcing every application to run a model API directly.

## Included in this release

- protocol specification in `spec/`
- example task packets in `examples/`
- browser-readable spec in `reference/`
- public landing page in `site/`
- repository governance and contribution files for public collaboration

## Highlights in v1.1

- clarified packet structure and task framing
- added stronger examples for plain text, JSON mode, retry packets, and stateful sessions
- separated public-facing documentation from the core specification
- prepared the repository for public review and proposals

## Status

**Public draft.**

AIP v1.1 is suitable for experimentation, reference implementations, discussion, and community feedback. It should be treated as a draft standard rather than a frozen final specification.

## Next step

Publish the repository, enable GitHub Pages, and invite feedback through issues and proposals.
