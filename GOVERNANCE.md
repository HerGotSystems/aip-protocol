# Governance

AIP v1.1 is published as a **public draft**.

## Current stewardship

- Steward: EMVY CHECK
- Status: founder-led draft stewardship
- Goal: move toward wider implementer feedback before freezing a stable 1.x line

## Change philosophy

Changes should improve one or more of:

1. interoperability
2. clarity
3. safety
4. implementation simplicity
5. model portability

## Versioning

- Patch-level editorial fixes do not change protocol meaning.
- Minor versions add compatible behavior, clarification, or registry growth.
- Major versions may revise packet semantics or compatibility guarantees.

## Path to wider stewardship

A future transition may introduce:
- a lightweight editor group
- public proposal workflow
- compatibility test cases
- reference validators and bridge implementations
